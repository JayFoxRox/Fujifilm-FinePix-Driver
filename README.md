# Installation

Written for Python 3.

**Requires:**

- [Pillow](https://pypi.org/project/Pillow/)
- [numpy](https://pypi.org/project/numpy/)
- [libusb1](https://pypi.org/project/libusb1/)
- [pyvirtualcam](https://pypi.org/project/pyvirtualcam/)
